Bootstrap-4 Template, Home-Page Slider for Zen Cart v1.5.7 and later

Version 1.0.1, by lat9

Current Support Thread at Zen Cart Forums: https://www.zen-cart.com/showthread.php?223910-ZCA-Bootstrap-4-Template-Support-Thread

GitHub repository: https://github.com/lat9/bs4_home_slider.

What it does

This sub-template provides a home-page slider (a.k.a. carousel) for the ZCA Bootstrap 4, v3.1.0 or later, template. You configure its output using Zen Cart's built-in Banner Manager and the 'pseudo-configuration' settings present in /includes/extra_datafiles/bs4_home_slider.php.

Files List

The following files are provided in the sub-template's distribution:

  1. /includes/extra_datafiles/bs4_home_slider.php
  2. /includes/languages/english/extra_definitions/bootstrap/bs4_home_slider_definitions.php
  3. /includes/templates/bootstrap/templates/tpl_index_categories.php, contains one change-section.
  4. /includes/templates/bootstrap/templates/tpl_index_default.php, contains one change-section.
  5. /includes/templates/bootstrap/templates/tpl_index_slider.php

Installation

This plugin provides a Bootstrap 4 styled slider for your Zen Cart's home-page. You should have previously installed:

  1. The 'base' ZCA Bootstrap 4 template, v3.1.0 or later: https://www.zen-cart.com/downloads.php?do=file&id=2191

Once those 'base' files have been installed, you're ready to install the 'sub-template' files.

Note:

  1. Unzip the slider's package file, which you've most likely already done if you're reading this!
  2. If you've made changes to the default Bootstrap 4 template's tpl_index_categories.php or tpl_index_default.php, you'll need to merge the slider-related changes (one section each) prior to the copying.
  3. Copy all the files present to your storefront, overwriting the 'base' bootstrap template's directories.
  4. If you've made a clone of the bootstrap template
    • Rename the bootstrap sub-directories in this sub-template's distribution to match the name of your "clone".
    • Copy (or merge) all the files to your clone also.
